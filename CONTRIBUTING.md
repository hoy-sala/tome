# Contributing to Tome

This is a personal fork of [bndct-devops/tome](https://github.com/bndct-devops/tome),
stripped down for my own use. I'm not actively seeking contributions, but if
you find a bug or have an idea, feel free to open an issue.

For the full-featured upstream project with KOReader sync, reading stats, and
more, head to [bndct-devops/tome](https://github.com/bndct-devops/tome). Issues
and PRs related to features that exist upstream should go there.
